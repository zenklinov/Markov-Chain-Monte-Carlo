# Markov-Chain-Monte-Carlo

This project is a simple implementation of the **Markov Chain Monte Carlo (MCMC)** method using Python. It aims to demonstrate how MCMC techniques‚Äîsuch as the Metropolis-Hastings algorithm‚Äîcan be used to sample from complex probability distributions, particularly when direct sampling is difficult or impossible.

## ‚ñ∂Ô∏è How to Run Locally

1. Clone this repository:

```bash
pip install -r requirements.txt
```
